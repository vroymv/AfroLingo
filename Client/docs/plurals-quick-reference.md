# Quick Reference: Swahili Plural Nouns

## The Four Main Noun Classes

### 1. M-WA Class üë• (People)

**Pattern**: M- ‚Üí WA-  
**Use**: People and living beings

| Singular   | Plural    | English          |
| ---------- | --------- | ---------------- |
| Mtu        | Watu      | Person/People    |
| Mtoto      | Watoto    | Child/Children   |
| Mwalimu    | Walimu    | Teacher/Teachers |
| Mwanafunzi | Wanafunzi | Student/Students |
| Mwanamke   | Wanawake  | Woman/Women      |
| Mwanaume   | Wanaume   | Man/Men          |

**Tip**: If it's a person, it likely uses M-WA!

---

### 2. M-MI Class üå≥ (Plants & Objects)

**Pattern**: M- ‚Üí MI-  
**Use**: Trees, plants, body parts, and some objects

| Singular | Plural  | English    |
| -------- | ------- | ---------- |
| Mti      | Miti    | Tree/Trees |
| Mkono    | Mikono  | Hand/Hands |
| Mguu     | Miguu   | Foot/Feet  |
| Mzigo    | Mizigo  | Load/Loads |
| Mlango   | Milango | Door/Doors |
| Mstari   | Mistari | Line/Lines |

**Tip**: Many body parts use M-MI class!

---

### 3. KI-VI Class üìö (Small Things)

**Pattern**: KI- ‚Üí VI-  
**Use**: Small objects, tools, languages, diminutives

| Singular | Plural  | English        |
| -------- | ------- | -------------- |
| Kiti     | Viti    | Chair/Chairs   |
| Kitabu   | Vitabu  | Book/Books     |
| Kisu     | Visu    | Knife/Knives   |
| Kiatu    | Viatu   | Shoe/Shoes     |
| Kikombe  | Vikombe | Cup/Cups       |
| Kioo     | Vioo    | Mirror/Mirrors |

**Tip**: Think "little things" - kitoto (small child), kijana (youth)

---

### 4. N-N Class üè† (No Change)

**Pattern**: SAME form for singular and plural  
**Use**: Animals, borrowed words, some objects

| Singular | Plural | English                    |
| -------- | ------ | -------------------------- |
| Nyumba   | Nyumba | House/Houses               |
| Ndege    | Ndege  | Bird/Birds or Plane/Planes |
| Samaki   | Samaki | Fish/Fish                  |
| Kalamu   | Kalamu | Pen/Pens                   |
| Nguo     | Nguo   | Cloth/Clothes              |
| Nchi     | Nchi   | Country/Countries          |

**Tip**: Context tells you if it's singular or plural!

---

## How to Identify Noun Classes

### Look at the Prefix

1. **M-** at the start:

   - Is it a person? ‚Üí M-WA class
   - Is it a plant/tree/object? ‚Üí M-MI class

2. **KI-** at the start:

   - ‚Üí KI-VI class (small things)

3. **N-** or other consonant at start:
   - ‚Üí Usually N-N class (no change)

### Practice Tips

1. **Learn in pairs**: Always learn singular with plural
2. **Group by class**: Study one class at a time
3. **Make flashcards**: Singular on one side, plural on the other
4. **Use in sentences**: Practice with complete sentences
5. **Listen to natives**: Pay attention to how plural forms sound

## Common Exceptions

### Irregular Plurals (Advanced)

Some nouns have irregular plural forms:

- Mke ‚Üí Wake (wife/wives)
- Rafiki ‚Üí Marafiki (friend/friends)
- Jicho ‚Üí Macho (eye/eyes)

_Don't worry about these yet - focus on the main patterns first!_

## Practice Exercises

### Exercise 1: Change to Plural

1. Mtu ‚Üí **\_** (person ‚Üí people)
2. Kiti ‚Üí **\_** (chair ‚Üí chairs)
3. Mti ‚Üí **\_** (tree ‚Üí trees)
4. Nyumba ‚Üí **\_** (house ‚Üí houses)

**Answers**: 1. Watu, 2. Viti, 3. Miti, 4. Nyumba

### Exercise 2: Identify the Class

1. Mtoto ‚Üí Watoto belongs to **\_** class
2. Kitabu ‚Üí Vitabu belongs to **\_** class
3. Mlango ‚Üí Milango belongs to **\_** class
4. Ndege ‚Üí Ndege belongs to **\_** class

**Answers**: 1. M-WA, 2. KI-VI, 3. M-MI, 4. N-N

2. Visu ‚Üí **\_** (knives ‚Üí knife)
3. Miguu ‚Üí **\_** (feet ‚Üí foot)
4. Kalamu ‚Üí **\_** (pens ‚Üí pen)

**Answers**: 1. Mwalimu, 2. Kisu, 3. Mguu, 4. Kalamu

## Glossary

- **Noun Class**: A grammatical category that groups nouns together
- **Prefix**: Letters at the beginning of a word (M-, KI-, etc.)
- **Singular**: One item (mtoto = one child)
- **Plural**: More than one (watoto = children)
- **Diminutive**: A small or younger version of something
