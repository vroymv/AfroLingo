# Vocabulary Images

This folder contains images for the vocabulary fill-in exercise in the alphabet lesson.

## Required Images

Each image should be:

- **Size**: 400x400 pixels (square format)
- **Format**: PNG with transparent background preferred
- **Style**: Simple, clear illustrations suitable for learning

### Image Files Needed

1. **pencil.png** - A pencil or pen
2. **notebook.png** - A notebook or composition book
3. **teacher.png** - A teacher (person teaching/at blackboard)
4. **backpack.png** - A school backpack
5. **eraser.png** - An eraser
6. **phone.png** - A mobile phone/smartphone

## Current Status

⚠️ **Images not yet added** - Currently showing placeholder icons. Add the actual images to this folder to enable proper display.

## Usage

These images are used in the vocabulary-fill-in activity where learners:

1. See the image
2. Fill in missing letters of the Swahili word
3. Hear the pronunciation via audio

The component will automatically use these images when they're placed in this folder.
