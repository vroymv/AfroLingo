# Alphabet Dictation Audio Instructions

## Required Audio File

**File Name:** `alphabet-dictation.mp3`

**Location:** `/assets/audio/alphabet-dictation.mp3`

## Audio Content

The audio file should contain a clear recording of someone pronouncing the following 10 Swahili alphabet letters in order:

**A, B, C, D, E, F, G, H, I, J**

## Recording Guidelines

1. **Pace:** Each letter should be pronounced clearly with a 1-2 second pause between letters
2. **Voice:** Use a clear, native or fluent Swahili speaker
3. **Quality:** Record in a quiet environment with minimal background noise
4. **Format:** MP3 format, 128kbps or higher
5. **Duration:** Approximately 15-20 seconds total

## Example Script

```
[Clear, steady pace]
A... B... C... D... E... F... G... H... I... J...
```

## Pronunciation Notes

- Pronounce each letter using Swahili pronunciation
- Maintain consistent volume throughout
- Leave enough pause between letters for learners to process

## Creating the Audio

You can:

1. Record it yourself using a voice recorder app
2. Use a text-to-speech service with Swahili support
3. Hire a voice actor on platforms like Fiverr
4. Extract from existing Swahili alphabet learning resources

## Testing

After adding the audio file:

1. Place it in `/assets/audio/alphabet-dictation.mp3`
2. Open the app and navigate to the alphabet lesson
3. Complete the first two activities
4. On activity 3/3, tap the "Play Audio" button
5. Verify the audio plays correctly
6. Test the answer checking by entering: A B C D E F G H I J
