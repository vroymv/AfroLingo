# Vocabulary Items Audio Guide

## Overview

This audio file should contain pronunciations of basic Swahili vocabulary words used in the alphabet lesson's vocabulary fill-in activity (Exercise 4).

## Required Words (in order)

1. **KALAMU** (pen/pencil)

   - Pronunciation: kah-LAH-moo

2. **DAFTARI** (notebook)

   - Pronunciation: dah-FTAH-ree

3. **MWALIMU** (teacher)

   - Pronunciation: mwah-LEE-moo

4. **MKOBA** (bag/backpack)

   - Pronunciation: m-KOH-bah

5. **KIFUTIO** (eraser)

   - Pronunciation: kee-foo-TEE-oh

6. **SIMU** (phone)
   - Pronunciation: SEE-moo

## Audio Format Requirements

- **File name**: `vocabulary-items.mp3`
- **Duration**: Approximately 15-20 seconds
- **Format**: MP3, 128kbps or higher
- **Content**: Each word should be pronounced clearly with a 1-2 second pause between words

## Recording Script

"Kalamu... Daftari... Mwalimu... Mkoba... Kifutio... Simu"

## Notes

- The audio is played when the user clicks the play button
- Users listen to the pronunciations while filling in the missing letters
- Clear, slow pronunciation is recommended for beginner learners
